# 全国保研院校热度地图｜云端自动抓取部署包

这个版本不要求你的电脑安装 Python。爬虫由 GitHub Actions 在云端自动运行，网页由 GitHub Pages 打开。

## 你要做的事

1. 注册或登录 GitHub。
2. 新建一个仓库，例如 `baoyan-heat-map`。
3. 把本文件夹里的所有文件上传到仓库。
4. 打开仓库 Settings → Pages。
5. Source 选择 `Deploy from a branch`，Branch 选择 `main`，Folder 选择 `/root`。
6. 等 1-3 分钟，GitHub 会给你一个网址。
7. 打开 Actions，手动运行“自动更新保研院校数据”。之后它会每周一自动更新。

## 数据更新逻辑

- 基础推免资格名单：自动抓研招网公开页面。
- 新增推免资格高校：自动尝试抓教育部/研招网公开通知。
- 推免率、推免名额、学院分配：不直接自动发布，先标记为待核验。

## 为什么这样做

推免率和名额分散在高校官网，有些是“本校推荐名单”，有些是“接收推免名单”，口径容易混。为了避免误导家长，必须进入待核验流程。
